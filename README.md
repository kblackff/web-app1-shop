# practical e-commerce web application

An e-commerce store having display of wears and clothings

This appilication is built with technologies Vite + React 
and a sever backend (SSR) using Node.js + Express that Vite supports

Uses Mongodb to provide database 
Integrates JSONWebToken for authentication


## features

It works about a login page to access the store and provides a legacy login user

provides a register page to create an account

provides payment or product purchase
